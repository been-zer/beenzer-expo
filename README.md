# BEENZER APP v1.0

Android: https://play.google.com/store/apps/details?id=beenzer.dapp

IOS: Comming soon (on Apple Store review)

## Project setup

Build app for both platforms:

```
eas build
```

Build app for android:

```
eas build:android
```

Build app for ios

```
eas build:ios
```

Download app -> https://beenzer.app
